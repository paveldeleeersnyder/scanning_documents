# Scan Documents

This is the repo for doing tests to see how well scanning documents works in flutter.  
This project contains four pages, each showing off a different package.
- document_camera_frame
- document_frame_scanner
- flutter_doc_scanner
- google_mlkit

Out of these test I believe flutter_doc_scanner is the best option for the OCR project.  
Most packages only work on android and IOS, and the google_mlkit_page will only work on android.

You can run this project using `flutter run`.
